# customer-support-ticket-triage
Week-1 Internship Project
#Project Description
This project is a Customer Support Ticket Triage System developed as part of a Week-1 Internship Project.
The system processes customer support tickets and helps in organizing them by cleaning the ticket text, identifying the issue type, assigning priority levels, and calculating SLA (Service Level Agreement) response time.
#Tools
Python – Core programming language
Google Colab – Code development and execution
Pandas – Data handling and analysis
GitHub – Project version control and submission
#Features
Load and explore customer support ticket data
Clean ticket description text
Classify tickets into issue types (LOGIN, PAYMENT, DELIVERY, REFUND, BUG, GENERAL)
Assign priority levels (P0–P3)
Calculate SLA response time in hours
Export final ticket report as a CSV file
#Output
Final CSV Report containing:
Cleaned ticket descriptions
Issue type
Priority level
SLA hours
